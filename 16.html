HTML 101: A Journey into Learning HTML

In the vast digital landscape of the internet, where every click leads to a new adventure, there lies a fundamental building block that powers the web: HTML, or HyperText Markup Language.

It doesn't matter if you’re a beginner and know absolutely nothing about HTML, I'm here to help every step of the way.

So, grab your virtual toolkit, in this case download spck editor on mobile, and let’s embark on this exciting journey!

What is HTML?

HTML is the backbone of web development. It’s a markup language that structures content on the web, allowing browsers to interpret and display text, images, links, and other multimedia elements.

So we're basically building a virtual home by simply writing what we want our home to be like... It's every bricklayers dream!.

Think of HTML as the skeleton of a website, or as your skeleton if you may; it provides the framework upon which everything else is built, or depends on. Without it, the internet would be a chaotic jumble of text and images, devoid of structure and meaning, just like our body would be without our skeleton.

1) The Basics: Your First HTML Page

Before diving into the intricacies of HTML, let’s create your very first web page. So you can either decide to pick up a pen and write this down or you can open a simple text editor (open the spck editor app) and type the following code:

  <!DOCTYPE html>
  <html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to my first web page.
    I’m learning HTML!</p>
  </body>
  </html>

If you're using spck editor, majority of this might have already been done for you.

Save this file with a .html extension (for example, index.html) as you'd see if you're using the spck editor app.

Now, open it in your web browser, or click the play button at the top right corner of your spck editor app, and voilà! You’ve just created your first web page!.

The <h1> tag creates a large heading, while the <p> tag defines a paragraph. This simple exercise is just the tip of the iceberg.

Understanding HTML Structure

Every HTML document follows a specific structure, which consists of several key components. This are like the cover of a book, without it we won't know what book it is.

1) Doctype Declaration:

The <!DOCTYPE html> declaration tells the browser that this document is an HTML5 document, which is the current version of HTML by the way.

2) HTML Element:

The <html> tag wraps all the content on the page.

3) Head Section:

The <head> section contains meta-information about the document, such as the title (displayed in the browser tab) and links to stylesheets or scripts.

4) Body Section:

The <body> section is where the visible content of the web page resides. This is where you’ll spend most of your time adding text, images, and other elements to create a website.

Diving Deeper: HTML Tags and Attributes

HTML is rich with tags that allow you to create various elements, but we don't really need to know all the tags to create a webpage. Here are some essential tags to get you started:

Headings:

We all know what headings are, they are the subject, topic or subtopic of something.

Use <h1> to <h6> for headings, with <h1> being the largest and <h6> the smallest.

Paragraphs:

The <p> tag is used for paragraphs of text.

Links:

The <a> tag creates hyperlinks. For example:

  
  <a href="https://www.example.com">
  Visit Example</a>.

Images:

The <img> tag displays images. For example:

 <img src="image.jpg" alt="Description  
 of image"

Lists:

Use <ul> for unordered lists and <ol> for ordered lists, with <li> for list items.

This are used for making a list of items, like a list of names, or a list of questions for a quiz website.

Attributes: The Spice of HTML

Attributes provide additional information about HTML elements. They are always included in the opening tag and come in name/value pairs.

For example, in the <a> tag, the href attribute specifies the URL the link points to. Here’s an example:

 <a href="https://www.example.com"
 target="_blank">Open Example in
  New Tab</a>

So what this does is, let's take for instance, the URL inside the href attribute is a link to our WhatsApp channel.

What this code does is, when "o Open Example in n New Tab" is clicked, we'll be taking to our channel in a new tab.

In this case, target="_blank" tells the browser to open the link in a new tab.

Styling with CSS: The Next Step

While HTML structures your content, CSS (Cascading Style Sheets) is what makes it visually appealing. Once you’re comfortable with HTML, learning CSS will allow you to style your web pages, control layouts, and create stunning designs.

Imagine transforming your plain text into a vibrant masterpiece with colors, fonts, and spacing!

Practice Makes Perfect

The key to mastering HTML (and any programming language) is practice. Start by creating simple web pages, experimenting with different tags and attributes.

And don't worry if you still don't get how it works, I'll be providing future updates to make sure you become a master and lover of coding!

But till then, keep practicing and stay active.
