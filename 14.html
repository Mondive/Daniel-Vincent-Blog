A Brief Introduction to HTML, CSS, and JAVASCRIPT

In a world of constant and continuous competition, being successful is no longer about who gets the highest grades in class or who knows more than who.

In present day world being successful is now based on the amount of people who value you and what you can do that can be useful to them.

And fairly enough learning how to code will enable you accomplish that and lots more if you take it seriously.

In this blog we will be learning the basics of HTML, CSS, and JAVASCRIPT.

1) HTML

HTML as you might know is an acronym for Hyper Text Markup Language.

It is a markup language which pretty much acts as the skeleton of a website.

This is where the necessary codes that defines the structure of the website are written. It is the foundation of a webpage.

Take for instance, we are building a robot.

Now before designing and giving instructions to the robot, we must first define the robot's features, like if we want it to have an arm, legs, one eye, or maybe two so it doesn't look like a cyclops.

Identifying and adding those features is pretty much what HTML is all about.

We first create a sketch of what want our website to have and include them in a code.

2) CSS

CSS stands for Cascading Style Sheet, a styling language. Picture it as we designing our robot.

Let's give the robot a name, Larry. Now we just finished giving larry his feature's.

He has a face, two arms, two legs, and he's already looking like a humanoid robot. The problem now is that Larry's looking ugly, really ugly.

That's where CSS comes to play. The job of CSS is to give beauty to a website, like it's colours, arrangements, formats... you get the gist.

So ehat CSS does in Larry's case is to simply make larry attractive and presentable, depending on who Larry's audience are.

3) JAVASCRIPT

Finally, JavaScript. And no! there is no full meaning of JavaScript, but you can simply call it JS.

Now JavaScript is a high level programming language, a human understandable language ehich can be compiled and interpreted to computer language.

JavaScript gives functionality to a website. Let's say we built larry to answer questions when asked. JavaScript role here is to receive that question and provide an answer to it.

Pretty cool huh? but mind you it's not that simple as larry would need to be fed a number of JavaScript codes for this to work all of which we have to type.(ever written 100+ lines of code before?).

So in summary

1) HTML stands for Hyper Text Markup Language and it gives the features of a website, like it's buttons, text and thing's like that.

2) CSS stands for Cascading Style Sheet, which is responsible for giving styles to a website, like the colour of the button, font style, arrangement of contents and so on.

3) And lastly JavaScript, which gives functionality to a website like what happens if a button is clicked.

Congratulations!! you've just covered the basics of HTML, CSS, and JAVASCRIPT.
